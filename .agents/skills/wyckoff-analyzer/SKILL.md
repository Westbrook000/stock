# 威科夫分析器 (Wyckoff Analyzer)

基于理查德·威科夫（Richard Wyckoff）的市场分析理论，对A股股票和ETF进行技术分析，判断当前阶段和进场时机。

> 参考：《威科夫操盘法》(孟洪涛著)、wyckoff_summary.md

## 威科夫理论核心

### 三大法则

| 法则 | 核心内容 | 应用 |
|------|----------|------|
| **供求关系** | 趋势动力来自供求不平衡 | 供过于求→下跌，供不应求→上涨 |
| **因果关系** | 趋势反转前需准备过程 | 吸筹是因，牛市是果；派发是因，熊市是果 |
| **努力与结果** | 成交量(努力)与价格(结果)应匹配 | 放量不涨=停止行为(SOT) |

### 四大阶段

```
┌─────────────────────────────────────────────────────────────────┐
│  积累(Accumulation) → 上涨(Markup) → 派发(Distribution) → 下跌(Markdown)  │
└─────────────────────────────────────────────────────────────────┘
```

| 阶段 | 说明 | 关键信号 | 量化指标 |
|------|------|----------|----------|
| **阶段A** | 趋势结束/盘整开始 | PS、SC、AR、ST | 价格下跌>10%，波动率>12% |
| **阶段B** | 吸筹区间 | 区间震荡、成交量萎缩 | 波动率<8%，成交量萎缩>40% |
| **阶段C** | 吸筹确认 | Spring、终极震仓 | 价格接近60日低点，MA20走平 |
| **阶段D** | 上涨/派发开始 | SOS、JOC、LPS | 放量长阳突破，MA20向上 |

## 关键信号详解

### 吸筹阶段信号

| 信号 | 英文 | 定义 | 交易意义 |
|------|------|------|----------|
| **PS** | Preliminary Support | 下跌后首次放量反弹 | 主力开始进场 |
| **SC** | Selling Climax | 超长阴线+超大成交量 | 恐慌抛售，主力接盘 |
| **AR** | Automatic Rally | SC后快速反弹 | 建立交易区间上沿 |
| **ST** | Secondary Test | 缩量回测SC低点 | 验证供应是否枯竭 |
| **Spring** | Spring | 跌破支撑后快速收回 | 吸筹确认信号 |
| **SOS** | Sign of Strength | 超长阳线+超大成交量 | 需求控制市场 |
| **JOC** | Jump Over Creek | 放量突破阻力位 | 跳离震荡区 |
| **LPS** | Last Point Support | 回调缩量 | 最后支撑点/加仓机会 |

### 派发阶段信号

| 信号 | 英文 | 定义 | 交易意义 |
|------|------|------|----------|
| **PSY** | Preliminary Supply | 上涨后放量回落 | 买力开始减弱 |
| **BC** | Buying Climax | 天量+新高+长上影 | 抢购高潮，主力出货 |
| **UT** | Upthrust | 假突破后快速回落 | 主力诱多陷阱 |
| **SOW** | Sign of Weakness | 放量跌破支撑 | 熊市确认 |
| **LPSY** | Last Point Supply | 最后派发点 | 派发完成 |

### 停止行为 (SOT)

```
努力 = 成交量（市场流入订单总量）
结果 = 价格进展幅度

放量 + 小波动 = 停止行为（关键信号）
  - 熊市：放量滞跌 → 需求接盘，可能见底
  - 牛市：放量滞涨 → 供应进场，可能见顶
```

## 趋势量化定义

### 趋势方向判断

| 趋势类型 | 量化标准 | 判断条件 |
|----------|----------|----------|
| **上涨趋势** | MA20 > MA60，且MA20向上 | 价格站上20日均线，且20日均线高于60日均线 |
| **下跌趋势** | MA20 < MA60，且MA20向下 | 价格在20日均线下方，且20日均线低于60日均线 |
| **横盘震荡** | 其他情况 | 价格在MA20和MA60之间波动 |

### 趋势强度

趋势强度计算：基于价格与均线的关系 + 20日价格变化
- 范围: -1 到 1
- > 0.3: 强上涨
- < -0.3: 强下跌

## 双周期验证

### 验证逻辑

| 规则 | 条件 | 得分 |
|------|------|------|
| 1 | 周线上涨 + 日线吸筹事件(SC/ST/Spring) | +2 |
| 2 | 周线上涨 + 日线上涨事件(SOS/JOC/LPS) | +1 |
| 3 | 周线下跌 + 日线派发事件(BC/UT/SOW) | -2 |
| 4 | 周线下跌 + 日线吸筹事件 | -1 |
| 5 | 周期共振(周线=日线且非震荡) | +1 |
| 6 | 趋势强度一致 | +1/-1 |

### 综合信号

| 得分 | 信号 | 建议 |
|------|------|------|
| >= 2 | 强信号 | 重点关注 |
| 1 | 中等信号 | 可考虑轻仓 |
| -1 | 偏弱信号 | 谨慎观望 |
| <= -2 | 风险信号 | 建议观望或做空 |

## 信号识别算法（概率云版本）

基于威科夫理论，使用概率云方法计算各事件的似然度(0-1)，置信度分层：高(>0.8)、中(0.6-0.8)、低(0.4-0.6)

### 1. 恐慌抛售 (SC) - 优化参数

```python
# 条件：
# - 近期下跌趋势(15日内跌幅>5%): 权重20%
# - 当日下跌(跌幅>2%): 权重25%
# - 放量(量比>1.3): 权重20%
# - 宽振幅或长下影: 权重15%

# 似然度 = 下跌趋势*0.20 + 当日下跌*0.25 + 放量*0.20 + 波动大*0.15 + 下影线*0.10
```

### 2. 自动反弹 (AR) - 优化参数

```python
# AR是SC之后的正常市场行为，不必太在意，不能在此抄底

# 条件：
# - 在SC后1-10天内: 权重35%
# - 反弹(收盘价高于开盘价): 权重25%
# - 成交量不过于萎缩(量比>0.6): 权重20%
# - 上涨: 权重20%

# 注意：AR只是反弹信号，需等待ST确认
```

### 3. 二次测试 (ST) - 优化参数

```python
# 成功的二次测试：蜡烛和成交量都必须小

# 条件：
# - 接近前期低点(±10%内): 权重30%
# - 成交量萎缩(<0.8): 权重30%
# - 振幅小(<8%): 权重20%
# - 在SC/AR之后5-25天内: 权重20%
```

### 4. 弹簧效应 (Spring) - 优化参数

```python
# 条件：
# - 跌破20日低点: 权重30%
# - 跌破幅度1-10%: 权重20%
# - 3日内收回: 权重30%
# - 缩量(量比<0.85): 权重20%

# Spring类型：
# - SPRING_BEST: 振幅<3%, 量比<0.5 (最佳弹簧)
# - SPRING_STRONG: 放量 + 收盘在50%之上 (强劲需求弹簧)
# - SPRING_CONFIRM: 需二次测试确认
# - SPRING_SHAKEOUT: 熊市陷阱

# 增强验证（detect_spring_enhanced）：
# - 市场背景过滤: 排除熊市背景
# - 需求跟随验证: Spring后N天内必须有需求跟随
# - 位置判断: JOC回测/吸筹底部/回调/普通
```

### 5. 强势出现 (SOS) - 优化参数

```python
# 条件：
# - 涨幅>4%: 权重20%
# - 放量(量比>1.8): 权重25%
# - 突破20日高点并收于高位: 权重25%
# - 连续3日上涨: 权重15%
# - 成交量放大: 权重15%
```

### 6. 抢购高潮 (BC) - 优化参数

```python
# 条件：
# - 前期明显上涨(20日涨幅>12%): 权重20%
# - 大阳线(涨幅>5%): 权重25%
# - 巨量(量比>2.2): 权重25%
# - 上影线明显: 权重15%
```

### 7. 上冲回落 (UT) - 优化参数

```python
# 条件：
# - 突破20日高点: 权重20%
# - 收盘在突破点下方(回落): 权重25%
# - 上影线>60%: 权重30%
# - 放量(量比>1.5): 权重25%
```

### 8. 弱势信号 (SOW) - 优化参数

```python
# 条件：
# - 跌破20日支撑: 权重30%
# - 放量下跌(量比>1.5): 权重25%
# - 明显跌幅(>2.5%): 权重25%
# - 前期已上涨: 权重20%
```

### 9. 最后支撑点 (LPS) - 优化参数

```python
# 条件：
# - 在SOS或BC之后5-20天: 权重20%
# - 明显缩量(<0.6): 权重30%
# - 小幅回调(3-8%): 权重25%
# - 振幅小(<4%): 权重25%
```

## 阶段判断（规则方法）

威科夫阶段判断使用规则方法（不依赖概率云），因为威科夫理论本身是基于明确的价格行为规则：

| 阶段 | 判断条件 |
|------|----------|
| **阶段A-下跌趋势** | 波动率≥12%，价格趋势向下 |
| **阶段B-吸筹区间** | 波动率<8%，成交量萎缩<60% |
| **阶段C-吸筹确认** | 有Spring或SC+ST成功 |
| **阶段D-上涨趋势** | 有SOS，波动率≥10% |
| **阶段E-派发/下跌** | 有BC或SOW |

## 威科夫分析事件

### 事件类型与日期

所有事件均包含以下信息：
- **日期**: 事件发生的具体日期
- **价格**: 事件发生时的价格
- **涨跌**: 相对前一交易日的涨跌幅
- **置信度**: 高/中/低
- **说明**: 事件描述

### 吸筹阶段信号

| 信号 | 英文 | 定义 | 交易意义 |
|------|------|------|----------|
| **PS** | Preliminary Support | 下跌后首次放量反弹 | 主力开始进场 |
| **SC** | Selling Climax | 超长阴线+超大成交量 | 恐慌抛售，主力接盘 |

| **SOS** | Sign of Strength | 超长阳线+超大成交量 | 需求控制市场 |

### 最佳进场位置

| 位置 | 信号 | 风险等级 |
|------|------|----------|
| Spring后 | 缩量Spring后放量上涨 | 低 |
| SOS回测 | 突破后缩量回调 | 低 |
