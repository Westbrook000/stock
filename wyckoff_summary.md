# 威科夫操盘理论

> 本文档基于孟洪涛《威科夫操盘法》提炼整理

## 核心原则

### 三大法则

| 法则 | 核心内容 | 应用 |
|------|----------|------|
| **供求关系** | 趋势动力来自供求不平衡 | 供过于求→下跌，供不应求→上涨 |
| **因果关系** | 趋势反转前需准备过程 | 吸筹是因，牛市是果；派发是因，熊市是果 |
| **努力与结果** | 成交量(努力)与价格(结果)应匹配 | 放量不涨=停止行为 |

### 聪明钱看盘顺序

1. **背景** - 当前处于哪个市场周期
2. **价量形态** - 蜡烛长度、成交量高度、变化速度
3. **形态性质** - 走势强弱、原因和后果
4. **结论预测** - 洞察主力意图，判断方向
5. **措施行动** - 进场、出场、收紧止损

## 市场周期详解

### 1. 积累阶段 (Accumulation)

熊市末端，主力建仓阶段。

#### 四大关键行为

| 行为 | 英文 | 特征 | 市场含义 |
|------|------|------|----------|
| 初步支撑 | PS | 下跌后首次放量反弹 | 主力开始进场 |
| 恐慌抛售 | SC | 超长阴线+超大成交量 | 公众恐慌出逃，主力接盘 |
| 自动反弹 | AR | SC后快速反弹 | 建立交易区间上沿 |
| 二次测试 | ST | 缩量回测SC低点 | 验证供应是否枯竭 |

#### 积累阶段细分

```
阶段一：熊市终止（PS → SC → AR → ST）
阶段二：区间交易（主力隐蔽吸筹）
阶段三：测试确认（Spring/震仓）
阶段四：牛市启动（SOS突破）
```

#### Spring（弹簧效应）

**定义**：价格快速突破支撑后迅速收回

**三种类型**：

| 类型 | 突破特征 | 后续操作 |
|------|----------|----------|
| 最佳Spring | 小蜡烛+小成交量 | 直接进场 |
| 需确认Spring | 略放量+中等蜡烛 | 等待二次测试 |
| 震仓Spring | 大蜡烛+大成交量 | 熊市陷阱，不可抄底 |

**Spring位置**：
- 牛市回调中的震荡区底部（最佳）
- JOC（跳离震荡区）后的回测
- 垂直需求柱底部回测
- 吸筹尾声的主力的洗盘

---

### 2. 上涨阶段 (Markup)

#### 关键事件

| 事件 | 英文 | 特征 | 含义 |
|------|------|------|------|
| 力量信号 | SOS | 超长阳线+超大成交量 | 需求控制市场 |
| 跳离区间 | JOC | 放量突破阻力 | 趋势确认 |
| 最后支撑 | LPS | 回调缩量 | 最后加仓机会 |

#### LPS识别要点

- 回调幅度小
- 成交量萎缩
- 不破前期关键支撑
- 阳线多于阴线

---

### 3. 派发阶段 (Distribution)

牛市末端，主力出货阶段。

#### 关键事件

| 事件 | 英文 | 特征 | 市场含义 |
|------|------|------|----------|
| 初步供给 | PSY | 上涨后放量回落 | 买力开始减弱 |
| 抢购高潮 | BC | 天量+新高+长上影 | 公众疯狂买入，主力出货 |
| 上冲回落 | UT | 假突破后快速回落 | 主力诱多陷阱 |
| 弱势信号 | SOW | 放量跌破支撑 | 熊市确认 |
| 破冰 | - | 跌破派发区支撑 | 最后派发点(LPSY) |

#### 派发两阶段

```
阶段一：警告阶段
- PSY：初步供给
- BC：抢购高潮

阶段二：确认阶段
- UT：上冲回落
- SOW：弱势信号
- 破冰：确认派发完成
```

---

### 4. 下跌阶段 (Markdown)

派发完成后进入下跌趋势。

## 量价关系详解

### 停止行为 (SOT)

**定义**：成交量放大但价格波动减小

| 市场环境 | 表现 | 含义 |
|----------|------|------|
| 熊市 | 放量滞跌 | 需求接盘，可能见底 |
| 牛市 | 放量滞涨 | 供应进场，可能见顶 |

### 量价配合表

| 价格 | 成交量 | 供求关系 | 后市判断 |
|------|--------|----------|----------|
| 上涨 | 放量 | 需求>供应 | 健康上涨 |
| 上涨 | 缩量 | 需求减弱 | 上涨乏力 |
| 下跌 | 放量 | 供应>需求 | 恐慌抛售 |
| 下跌 | 缩量 | 供应枯竭 | 正常回调 |

### 努力与结果

```
努力 = 成交量（市场流入订单总量）
结果 = 价格进展幅度

匹配情况：
✓ 放量 + 大涨 = 需求强劲
✓ 缩量 + 小跌 = 供应枯竭
✗ 放量 + 小波动 = 停止行为（关键信号）
```

## 实战交易要点

### 最佳进场位置

| 位置 | 信号 | 风险等级 |
|------|------|----------|
| Spring后 | 缩量Spring后放量上涨 | 低 |
| SOS回测 | 突破后缩量回调 | 低 |
| LPS | 牛市回调缩量 | 中 |
| 二次测试成功 | ST成交量大幅萎缩 | 低 |

### 止损设置

| 进场位置 | 止损位置 |
|----------|----------|
| Spring | SC低点下方1-2% |
| SOS | 突破点下方3% |
| LPS | 支撑下方3-5% |

### 危险信号

| 信号 | 含义 | 行动 |
|------|------|------|
| 连续失败Spring | 牛市动能减弱 | 准备离场 |
| UT上冲回落 | 派发开始 | 考虑做空 |
| SOW弱势信号 | 熊市确认 | 立即离场 |
| 破冰 | 派发完成 | 做空 |

## 常见陷阱与对策

### 陷阱1：熊市中抄底Spring

**错误**：不看背景，看到Spring就进场
**对策**：熊市中的Spring是陷阱，必须等待二次测试

### 陷阱2：追突破

**错误**：看到突破就买入
**对策**：确认是真突破(SOS)还是假突破(UT)

### 陷阱3：忽视成交量

**错误**：只看价格形态
**对策**：成交量是验证信号的关键

### 陷阱4：单一周期分析

**错误**：只看日线
**对策**：周线定方向，日线找信号，60分钟找进场
